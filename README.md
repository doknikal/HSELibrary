# HSE Library

Приложение в строке меню (menubar) для macOS, которое в один клик включает доступ
к электронным ресурсам библиотеки ВШЭ через прокси.

## Установка

1. Скачайте `HSE_Library.dmg` со страницы **[Releases](../../releases)**.
2. Откройте `.dmg` и перетащите **HSE Library** в папку «Программы».

## Первый запуск

Приложение распространяется без платной подписи Apple, поэтому при первом запуске
macOS попросит подтверждение:

- Кликните по приложению **правой кнопкой** → **Открыть** → в диалоге ещё раз **Открыть**.
- Либо: «Системные настройки» → «Конфиденциальность и безопасность» → кнопка **«Открыть всё равно»**.

Если появляется сообщение, что приложение «повреждено и не может быть открыто»,
выполните в Терминале одну команду и запустите снова:

```bash
xattr -dr com.apple.quarantine "/Applications/HSE Library.app"
```

## Использование

1. В строке меню появится значок **ВШЭ**.
2. **Войти…** — введите логин и пароль ВШЭ (хранятся в Связке ключей macOS).
3. **Включить прокси** — готово, ресурсы ВШЭ открываются как из кампуса.
4. **Проверить доступ** — проверка, что всё работает.

При выходе из приложения прокси автоматически отключается.

## Требования

macOS 13 и новее · Intel и Apple Silicon.
